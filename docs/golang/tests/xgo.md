## 简介

xgo 项目是 Go 语言的跨平台构建工具，它允许您使用单个命令构建和测试您的 Go 代码，而无需在每个平台上手动安装 Go。

## 使用说明

- 测试单个方法

```sh
xgo test -v -run Test_Method_Name ./
```
